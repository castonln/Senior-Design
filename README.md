## Project Description
Crosley Tower Defense is a tower defense game set on UC's campus. The objective is to defend the now ill-fated Crosley Tower from demolition. Players can strategically place students to protect the tower from waves of Triceracopters (as found in Langsam Library) who remove the tower floor by floor. The goal of the project is to create a game that is a topical and fun way to bolster school spirit among students, especially incoming 2026 freshmen.

## How to play
Players start with a fully constructed **tower** featuring a grid and a starting amount of **points.**

Click the **"Shop"** button to open the game's shop. **Students** can be hired to defend the tower for a set amount of **points.** A **student,** when bought, can be placed on a grid cell. Outer cells are for defending against enemies, **Triceracopters,** while inner cells are for support-based students who don't attack, but instead help in other ways.

There are *four* **lanes,** one for each protrusion of the **tower's** top.

When players are ready, click the **"Start Wave"** button to begin the **wave.** A **wave** is a collection of **Triceracopters** that attack over a period of time. Once all the **Triceracopters** are defeated, the **wave** is over, and players have time to prepare for the next wave.

Each time a **Triceracopter** is defeated, **points** are awarded to the player, which can be used in the shop.

**Students** can be moved to a different grid cell at any time. Simply click or tap a student and then tap a different cell. This allows you to move a **student** around to defend different **lanes** in real time depending on the needs of the **wave.**

If you wish to use different **students** for different **waves,** you can sell a student by tapping on them and pressing the **"Sell"** button.

## FAQ

*When does the game end?*
The game is intended to be infinite or end when the tower is fully destroyed. Try to get the highest score you can!

*Can I move students after they are placed?*
Yes, students are not permenantly bound to one location. See the **How to play** guide.

*Can waves be sped up / slowed down?*
No. Waves are set at a particular speed since a main aspect of the game is the ability to move students in real time to defend different areas.
